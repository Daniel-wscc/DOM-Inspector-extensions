# DOM Inspector Pro - Chrome 擴展

一個功能強大的 Chrome 擴展，專門用於檢查網頁上的 DOM 元素，特別針對圖片顯示原始尺寸和檔名。

## 功能特色

### 🖼️ 圖片專用功能
- **原始尺寸顯示**: 顯示圖片的原始寬高（像素）
- **檔名提取**: 從 URL 中提取並顯示圖片檔名
- **顯示尺寸**: 顯示圖片在頁面上的實際顯示尺寸
- **來源資訊**: 顯示圖片的完整 URL 和 Alt 文字

### 🔍 通用 DOM 檢查
- **元素高亮**: 滑鼠懸停時高亮顯示目標元素
- **即時資訊**: 懸停時顯示元素的基本資訊
- **詳細檢視**: 點擊元素查看完整的詳細資訊
- **樣式分析**: 顯示計算後的 CSS 樣式

### 🎯 使用者體驗
- **直觀介面**: 簡潔美觀的彈出視窗設計
- **一鍵切換**: 單擊按鈕即可開啟/關閉檢查模式
- **視覺回饋**: 清晰的狀態指示和操作提示
- **響應式設計**: 支援不同螢幕尺寸

## 安裝方法

### 方法 1: 開發者模式安裝
1. 下載或克隆此專案到本地
2. 開啟 Chrome 瀏覽器，前往 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇此專案的資料夾
6. 擴展安裝完成！

### 方法 2: 打包安裝
1. 在開發者模式下，點擊「封裝擴充功能」
2. 選擇專案資料夾
3. 點擊「封裝擴充功能」
4. 下載生成的 `.crx` 檔案
5. 拖拽到 Chrome 擴展頁面安裝

## 使用方法

### 基本操作
1. **啟動檢查**: 點擊工具列中的擴展圖標
2. **開啟檢查模式**: 在彈出視窗中點擊「啟動檢查模式」
3. **檢查元素**: 滑鼠懸停在網頁元素上查看資訊
4. **詳細檢視**: 點擊元素查看完整詳細資訊
5. **關閉模式**: 再次點擊按鈕關閉檢查模式

### 圖片檢查
- 懸停在圖片上會顯示：
  - 檔名
  - 顯示尺寸
  - 原始尺寸
  - 來源 URL
  - Alt 文字

### 其他元素檢查
- 懸停時顯示：
  - 元素標籤
  - 尺寸資訊
  - 類別名稱
  - ID 屬性
  - 文字內容預覽

## 開發說明

### 本地開發
1. 修改程式碼後，在擴展頁面點擊「重新載入」
2. 重新整理測試網頁
3. 使用 Chrome DevTools 除錯

### 除錯技巧
- 在 `content.js` 中使用 `console.log()` 輸出除錯資訊
- 檢查 Chrome DevTools 的 Console 和 Network 標籤
- 使用擴展頁面的「檢查檢視」功能

## 常見問題

### Q: 擴展無法啟動？
A: 檢查是否開啟了開發人員模式，並確認所有檔案路徑正確。

### Q: 檢查模式沒有反應？
A: 重新載入擴展和網頁，確認內容腳本已正確載入。

### Q: 圖片資訊顯示不完整？
A: 某些圖片可能因為 CORS 限制無法獲取完整資訊。
